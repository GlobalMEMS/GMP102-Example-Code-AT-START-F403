AT-START-F403 + GMP102 example code
===================================

Requirements
-----------
- AT-START-F403 development board
- Sensor Fusion Arduino Daughter Board V1.0: GMP102 is on the sensor board

I2C Connections
---------------
- Use I2C1
  - SCL: PB8
  - SDA: PB9
- GMP102 I2C 7-bit slave address: 0x6C
